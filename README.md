# ESII-Projects
Mini projects for the Embedded Systems II course.

## Compile
```bash
$ make
```

## Run
```bash
$ make run
```

## Clean
```bash
$ make clean   //remove object files
$ make nuke    //remove object files and executables
```

## File structure
  src:      source code cpp files  
  include:  header files   
  build:    object files  
  bin:      executables
